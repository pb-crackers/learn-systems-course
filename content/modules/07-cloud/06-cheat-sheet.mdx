## Service Mapping Reference

| Layer | Docker / Linux | AWS | GCP | Azure |
|-------|---------------|-----|-----|-------|
| Compute (IaaS) | docker run | EC2 | Compute Engine | Azure VMs |
| Compute (PaaS) | docker compose | ECS Fargate | Cloud Run | Container Apps |
| Serverless | (no equivalent) | Lambda | Cloud Functions | Azure Functions |
| Network | docker network | VPC | VPC Network | Virtual Network |
| Subnet | --subnet | Subnet | Subnet | Subnet |
| Load Balancer | nginx upstream | ALB | Cloud Load Balancing | Azure Load Balancer |
| Object Storage | (no equivalent) | S3 | Cloud Storage | Azure Blob Storage |
| Block Storage | docker volume | EBS | Persistent Disk | Azure Disk Storage |
| File Storage | NFS mount | EFS | Filestore | Azure Files |
| Identity | Linux user | IAM User | Cloud IAM | Azure AD User |
| Permission | chmod/chown | IAM Policy | IAM Role | RBAC Role |
| Role Assume | sudo | IAM Role Assume | Service Account | Managed Identity |
