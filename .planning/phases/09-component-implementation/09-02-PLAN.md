Create `components/content/ScenarioQuestion.tsx` — a client component that renders a scenario-contextualized question between exercise steps with an expandable answer reveal.

Component contract:
```typescript
interface ScenarioQuestionProps {
  question: string  // Question text that references the exercise scenario
  answer: string    // Expected answer, revealed on click
}
```

This component needs `'use client'` because it uses useState for the expand/collapse state.

Rendering structure:
1. **Question block**: A visually distinct block (similar to Callout styling) that presents the question. Use a left border accent in a distinct color (blue/purple — something different from Callout's green/amber/blue to avoid confusion). Include an icon — use `HelpCircle` from lucide-react (or `MessageCircle` — pick whichever feels more "think about this" than "error/warning").

   Layout:
   - Icon + label "Think About It" as a header line (text-sm font-semibold)
   - Question text below (text-sm, normal prose)
   - A "Show Answer" / "Hide Answer" toggle button at the bottom

2. **Answer reveal**: When the learner clicks "Show Answer":
   - Show the answer text in a slightly indented block below the question
   - Use the same expandable pattern as VerificationChecklist hints: `text-sm text-muted-foreground mt-2 pl-3 border-l-2 border-border`
   - ChevronDown icon that rotates 180 degrees when expanded (same pattern as VerificationChecklist)
   - Toggle button text changes to "Hide Answer" when expanded

Design intent from CONTEXT.md: "I am running this command so I can answer THIS question" — the question should feel like a natural part of the exercise flow, purpose-driven, not quiz-like. The visual treatment should be lighter/softer than a Callout (which signals important information) — this is a thinking prompt.

Styling specifics:
- Container: `my-4 rounded-lg border border-border/50 bg-muted/10 px-4 py-3` (lighter than Callout, blends with exercise flow)
- Left border accent: `border-l-4 border-l-violet-500/60` (violet distinguishes from Callout green/amber/blue)
- Header: flex items-center gap-2 mb-2, icon h-4 w-4, "Think About It" text-sm font-semibold text-violet-400
- Question: text-sm leading-relaxed
- Toggle button: text-xs text-muted-foreground hover:text-foreground flex items-center gap-1 mt-2
- Answer area: mt-2 pl-3 border-l-2 border-border text-sm text-muted-foreground

If question is empty/undefined, return null.
